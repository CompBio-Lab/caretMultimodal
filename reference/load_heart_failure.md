# Load the example heart failure datasets

Loads example datasets included with the package into the user's
environment.

## Usage

``` r
load_heart_failure()
```

## Value

Invisibly returns the names of the objects loaded.

## References

Improved Prediction of Cardiac-Related 3-Month Hospitalizations: A Pilot
